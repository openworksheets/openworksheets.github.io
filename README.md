# WorkPDF

WorkPDF es una aplicación web para convertir PDFs o imágenes en fichas interactivas autocorregibles, de forma parecida a TopWorksheets. El profesorado puede subir un documento, colocar encima distintos tipos de campos de respuesta y configurar las soluciones, la puntuación, las opciones de corrección y las restricciones de acceso.

Permite crear actividades con respuestas cortas, respuestas numéricas, opción única o múltiple, verdadero/falso, desplegables, huecos, emparejamientos, ordenaciones, arrastrar a zonas y unir con flechas. También incluye elementos de diseño como texto, imágenes, líneas, flechas y formas.

La ficha se exporta como un archivo ZIP que contiene todo lo necesario. Después puede subirse a Google Drive u otro alojamiento público y compartirse con el alumnado mediante un enlace. Los alumnos responden desde el navegador, reciben la corrección si el profesor lo permite y generan una entrega en JSON que puede verificarse posteriormente.

Las fichas pueden configurarse para abrirse o cerrarse cuando se quiera, indicando una fecha y hora de inicio y de finalización. También se pueden añadir otras condiciones de uso, como tiempo límite por intento, contraseña de acceso, entrega automática al finalizar el plazo o número máximo de intentos.

Funciona sin servidor, sin cuentas y sin instalaciones, por lo que es especialmente útil para prácticas, deberes y actividades de autoevaluación. Las restricciones de acceso se aplican en el navegador del alumno, así que ayudan a organizar la actividad, pero no sustituyen a un sistema de examen de alta seguridad.
